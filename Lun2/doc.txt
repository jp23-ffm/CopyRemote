● Voilà. Usage :

  python manage.py compact_server_history --dry-run --zone weekly --report C:\Temp\report_weekly.html
  python manage.py compact_server_history --dry-run --zone all    --report C:\Temp\report_all.html

  Le fichier HTML généré contient :

  En-tête : stats globales (total paires / compactées / skippées / del / ins / net), badge DRY RUN si applicable.

  Table filtreable :
  - Recherche par SERVER_ID ou APP
  - Filtre par zone (Weekly / Monthly)
  - Case "Inclure les skips"
  - 50 paires par page
  - Colonnes : Del (rouge) / Ins (vert) / Net / mini-timeline avant (barres colorées) / mini-timeline après

  Clic sur une ligne → déplie un panneau avec :
  - SVG avant/après (comme notre viz, mais avec les vraies données)
  - Deux tables détaillées (avant / après) avec les 5 champs display
  - Cellules surlignées en jaune quand la valeur change entre deux rows consécutifs
  - Valeur open ∞ en vert pour les rows sans valid_to

  La couleur des barres est dérivée du hash des attrs — même couleur = même état, ce qui rend la compression
  visuellement évidente (les segments de même couleur qui se fondent en un seul dans l'après).
