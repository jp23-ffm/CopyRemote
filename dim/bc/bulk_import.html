Bulk CSV Import

    Upload a CSV file to perform a bulk update. The format is auto-detected from the header row.

    Supported formats

    Server Import

    Updates server fields. hostname_unique is mandatory. Other fields are optional (empty = unchanged).

    • hostname_unique (mandatory): Server hostname
    • priority_asset: Priority Asset (YES/NO)
    • in_live_play: Live Play Status (YES/NO)
    • action_during_lp: Action during the Live Play
    • original_action_during_lp: Original Action during the Live Play
    • cluster: Cluster (YES/NO)
    • cluster_type: Cluster Type
    hostname_unique;priority_asset;in_live_play;action_during_lp;original_action_during_lp;cluster;cluster_type
