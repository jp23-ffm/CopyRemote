  Sur l'env DEV (SQLite) :
  # --output évite de charger tout en mémoire (streaming vers fichier)
  python manage.py dumpdata inventory.ServerHistory --indent 0 --output server_history.json
  python manage.py dumpdata inventory.FieldSnapshotFiltered --indent 0 --output field_snapshots.json

  Sur l'env cible (Postgres) :
  # S'assurer que les tables sont vides (sinon conflits de PK)
  python manage.py shell -c "from inventory.models import ServerHistory, FieldSnapshotFiltered;
  ServerHistory.objects.all().delete(); FieldSnapshotFiltered.objects.all().delete()"

  python manage.py loaddata server_history.json
  python manage.py loaddata field_snapshots.json

  loaddata gère tout seul la conversion SQLite → Postgres (types, NULL, etc.).
