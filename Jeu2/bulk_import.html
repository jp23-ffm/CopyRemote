Bulk Import of Servers Information Using a CSV

    Upload a CSV file to update or create server annotations.

    The CSV file MUST use semicolon as separator, MUST have the format below, and 'hostname' MUST at least be filled:

    • 'hostname' (mandatory): Server hostname
    • 'type': Type of annotation (e.g. INFO, WARNING, ERROR)
    • 'servicenow': ServiceNow ticket number
    • 'notes': Annotation text
    • 'action' (optional): add (default) to add a new annotation, or resolve to resolve the latest active annotation

    Example — adding annotations (4 columns, backward compatible):

    hostname;type;servicenow;notes
server1;INFO;RITM123456;In ME until 24th
server2;WARNING;INC123456;OPENSVC Cluster issue
server3;PENDING;RITMXXXXXX;To be checked

    Example — mixed add/resolve (5 columns with action):

    hostname;type;servicenow;notes;action
server1;;;;resolve
server2;WARNING;RITM456;New issue detected;add
